    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h2><a id="user-content-getting-and-cleaning-data-course-project" class="anchor" href="#getting-and-cleaning-data-course-project" aria-hidden="true"><span class="octicon octicon-link"></span></a>Getting and Cleaning Data Course Project</h2>

<p>The goal of this project is to create a clean dataset from data collected from the accelerometers from the Samsung Galaxy S smartphone. A full description is available at the site where the data was obtained:</p>

<p><a href="http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones">http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones</a></p>

<p>Here are the data for the project:</p>

<p><a href="https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip">https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip</a></p>

<p>The <strong>run_analysis.R</strong> R script contains the R code that does the following:</p>

<ol>
<li>Downloads and unzips the original data.</li>
<li>Reads and merges the training and testing datasets.</li>
<li>Sets the names for features as described in <em>features.txt</em>.</li>
<li>Extracts the variables describing mean and standard deviation.</li>
<li>Replaces activity ids with proper names as described in <em>activity_labels.txt</em></li>
<li>Reshapes the data to create a dataset with the average of each variable for each activity and each subject.</li>
<li>Writes this dataset into <em>tidyData.txt</em> file.</li>
</ol>

<p>The <strong>codebook.md</strong> file describes the output dataset, which is stored in <strong>tidyData.txt</strong> file.</p>
